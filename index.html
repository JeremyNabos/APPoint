APPoint

APPoint is an appointment system used by the Marinduque State College. It can be used by both student and teachers to book an appointment before coming to school to prevent large crowds from gathering.

About School

Marinduque State College (MSC) is a Philippine state university. The institution, which was founded in 1952, has a main campus on Boac as well as three branch campuses. Agriculture, Science, Technology, and Trades programs are available at this college.
